Using the Figma Libraries#

To begin using a Match Figma Library, select "Libraries" from the menu in your desired project. Use the toggle to turn on/off any desired libraries. We recommend turning on all the libraries (tokens, icons, and components) that exist for your theme.

Once you've enabled the libraries you can add components in the "Assets" panel. Tokens can be added using the menus on the right side of the UI. For more specific information on adding components, tokens, and icons reference the corresponding Match project. If you run into any issues or have questions don't hesitate to come to reach our to us on slack!
