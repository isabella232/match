👋 Hi Twilio developer. Follow the steps below to begin building with Match

1.Install external dependencies

PackageVersion
react^16.8.6
react-dom^16.8.6
prop-types^16.8.6
react-uid^2.2.0

Install dependencies using yarn

yarn add react react-dom prop-types react-uid

2. Install Match packages

You can consume Match’s components, icons and tokens packages in order to build custom page sections.

PackageVersion
@twilio-labs/match-tokens0.1.1
@twilio-labs/match-styling-library0.1.1

Install Match packages using yarn

yarn add @twilio-labs/match-tokens @twilio-labs/match-styling-library
