👋 Hi Twilio developer. Follow the steps below to begin building with Match

1.Install external dependencies

PackageVersion
react^16.8.6
react-dom^16.8.6
prop-types^16.8.6
react-is^16.13.1

Install dependencies using yarn

yarn add react react-dom prop-types react-is styled-components

2. Install Match packages

You can consume Match’s components, icons and tokens packages in order to build custom page sections.

PackageVersion
@twilio-labs/match-tokens0.1.0
@twilio-labs/match-themes0.1.0

Configure GitHub Package Regsitry

The Match packages are hosted on the GitHub Package Registry and require some additional configuration to install.

For Yarn 2 and up, add the following to the .yarnrc.yml file in the root of your project:

npmScopes:
  twilio-labs:
    npmRegistryServer: "https://npm.pkg.github.com"

For Yarn 1, add the following to the .yarnrc file in the root of your project.

"@twilio-labs:registry" "https://npm.pkg.github.com"

For NPM, add the following to the .npmrc file in the root of your project.

@twilio-labs:regsitry=https://npm.pkg.github.com

Install Match packages

With Yarn:

yarn add @twilio-labs/match-tokens @twilio-labs/match-themes

With NPM:

npm install --save @twilio-labs/match-tokens @twilio-labs/match-themes
