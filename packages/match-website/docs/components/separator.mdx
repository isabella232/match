## Guidelines

### About Separators

The separator can be used to help visually divide two groups of menu items in a menu or to provide a horizontal rule between two sections of a page.

### Accessibility

Note: The plain divider in gray20 is not color contrast accessible and is intended for visual embellishment. Use it paired with white space to create enough visual space between elements.

Use the separator to help separate page sections with differing content

### Primary

The primary variant is the default, and should be used on areas where the dominant background color is close to white.

```jsx live

```

### Inverse

The inverse variant should be used on areas where the dominant background color is dark
