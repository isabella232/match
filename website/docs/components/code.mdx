The Code component visually distinguishes inline text as a fragment of computer code.

**Sources:** [Github](https://github.com/twilio-labs/match/tree/main/packages/match-components/src/code)

## Usage Guide

### Installation

```shell
yarn add @twilio-labs/match
```

### Usage

```jsx
import { Code } from "@twilio-labs/match/components";

const Component = () => <Code>ahoy()</Code>;
```

### Code Props

<Props of="Code" />

## Guidelines

### About Code

The Code component emphasizes short, inline pieces of computer code such as variable names, commands and function calls. For multiline code samples and bash shell prompts, see the [Snippet component](./snippet.mdx).
