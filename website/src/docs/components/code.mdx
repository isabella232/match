## Code

### Usage

```jsx
import { Code } from "@twilio-labs/match/components";

const Component = () => <Code>ahoy()</Code>;
```

### Props

| Prop     | Type        | Default | Description     |
| -------- | ----------- | ------- | --------------- |
| variant? | CodeVariant | 'dark'  | 'dark', 'light' |

---

## Guidelines

### About Code

The Code component emphasizes short, inline pieces of computer code such as variable names, commands and function calls. For multiline code samples and bash shell prompts, see the [Snippet component](./snippet.mdx).
