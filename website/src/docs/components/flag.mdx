## Code

### Usage

```jsx
import { Flag, flagCodes } from "@twilio-labs/match/components";

const Component = () => <Flag code={"US"} />;
```

### Props

| Prop        | Type     | Default  | Description                                                                                            |
| ----------- | -------- | -------- | ------------------------------------------------------------------------------------------------------ |
| code        | string   | null     | The specified flag code, see https://flagpack.xyz/docs/flag-index for reference                        |
| size?       | FlagSize | 'normal' | 'small', 'normal', 'large'                                                                             |
| decorative? | boolean  | false    | Determines if icon is decorative or informative. Decorative icon titles are not read by screenreaders. |

---

## Examples

### Decorative Icons

`alt` text is provided for accessibility compliance unless the icon is specifically designated as decorative.

```jsx live
<Flag code={"US"} decorative={false} />
```
