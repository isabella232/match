## Margin Props

Most Match components support margin props. These can be any valid [spacing token](/design-tokens/tokens-list/#space).

| Prop          | Description                    |
| ------------- | ------------------------------ |
| padding       | Padding on all 4 sides.        |
| paddingX      | Padding on the left and right. |
| paddingY      | Padding on the top and bottom. |
| paddingTop    | Padding on the top.            |
| paddingRight  | Padding on the right.          |
| paddingBottom | Padding on the bottom.         |
| paddingLeft   | Padding on the left.           |
