## 1. Install external dependencies

| Package                                                 | Version |
| ------------------------------------------------------- | ------- |
| [formik](https://formik.org)                            | 2.x     |
| [prop-types](https://github.com/facebook/prop-types)    | 15.x    |
| [react](https://reactjs.org/)                           | 17.x    |
| [react-dom](https://reactjs.org/docs/react-dom.html)    | 17.x    |
| [react-is](https://www.npmjs.com/package/react-is)      | 17.x    |
| [react-uid](https://www.npmjs.com/package/react-is)     | 2.x     |
| [styled-components](https://www.styled-components.com/) | 5.x     |
| [styled-system](https://styled-system.com/)             | 5.x     |

```shell
npm install --save formik prop-types react react-dom react-is react-uid styled-components styled-system
```

## 3. Wrap your app with a theme component

For example, if you want to use the `TwilioTheme`:

```jsx
import * as React from "react";
import { TwilioTheme } from "@twilio-labs/match/themes";

const App: React.FC = ({ children }) => <TwilioTheme>{children}</TwilioTheme>;
export { App };
```

See [available themes here](https://github.com/twilio-labs/match/blob/main/packages/match-themes/src/constants.ts).

## Code Quality

Match distributes reusable configuration for [ESLint](https://eslint.org/), [Stylelint](https://stylelint.io/), [Browserslist](https://github.com/browserslist/browserslist) and [TypeScript](https://www.typescriptlang.org/). It is highly recommended to extend from these configs for a consistent development experience. Additionally we recommend using [cspell](https://github.com/streetsidesoftware/cspell/tree/main/packages/cspell) and [Prettier](https://prettier.io/) for code spell checking and automatic formatting.

### ESLint

Install the Match ESLint configuration and all peer dependencies.

```bash
npm install --save-dev @twilio-labs/eslint-config-match @typescript-eslint/eslint-plugin @typescript-eslint/parser babel-eslint eslint eslint-config-prettier eslint-plugin-eslint-comments eslint-plugin-import eslint-plugin-jest eslint-plugin-jest-formatting eslint-plugin-jsx-a11y eslint-plugin-promise eslint-plugin-react eslint-plugin-react-hooks eslint-plugin-testing-library eslint-plugin-unicorn
```

Create a `.eslintrc.json` file in the root of your project and add the following.

```json
{
  "extends": ["@twilio-labs/eslint-config-match"]
}
```

### Stylelint

Install the Match Stylelint configuration and all peer dependencies.

```bash
npm install --save-dev @twilio-labs/stylelint-config-match stylelint stylelint-a11y stylelint-config-prettier stylelint-config-rational-order stylelint-config-recommended stylelint-declaration-block-no-ignored-properties stylelint-no-unsupported-browser-features stylelint-order
```

Create a `.stylelintrc.json` file in the root of your project and add the following.

```json
{
  "extends": ["@twilio-labs/stylelint-config-match"]
}
```

### TypeScript

Install the Match TypeScript configuration.

```bash
npm install --save-dev @twilio-labs/tsconfig-match
```

Create a `tsconfig.json` file in the root of your project and add the following.

```json
{
  "extends": "@twilio-labs/tsconfig-match"
}
```

### Browserslist

Install the Match browserslist configuration.

```bash
npm install --save-dev @twilio-labs/browserslist-config-match
```

Add the following to your project's `package.json` file.

```json
{
  "browserslist": ["extends @twilio-labs/browserslist-config-match"]
}
```
