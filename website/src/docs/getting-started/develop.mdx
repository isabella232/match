## 1. Install external dependencies

| Package                                                 | Version |
| ------------------------------------------------------- | ------- |
| [formik](https://formik.org)                            | 2.x     |
| [prop-types](https://github.com/facebook/prop-types)    | 15.x    |
| [react](https://reactjs.org/)                           | 17.x    |
| [react-dom](https://reactjs.org/docs/react-dom.html)    | 17.x    |
| [react-is](https://www.npmjs.com/package/react-is)      | 17.x    |
| [react-uid](https://www.npmjs.com/package/react-is)     | 2.x     |
| [styled-components](https://www.styled-components.com/) | 5.x     |
| [styled-system](https://styled-system.com/)             | 5.x     |

```shell
npm install --save formik prop-types react react-dom react-is react-uid styled-components styled-system
```
